# Quick Start

1. Copy SKILL.md into your Hermes skills directory.
2. Copy templates/ and prompts/ into your own workflow repo.
3. Read INSTALL.md for the manual install steps and job setup.
4. Enable the `workflow-validator` and `workflow-smoke-test` jobs.
5. Point the workflow at an Obsidian vault.
6. Use the workflow index as the quick-scan source of truth.
