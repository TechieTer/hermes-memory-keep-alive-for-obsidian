# Task Note Template

## RESUME.md
- Task name
- Current status
- Last heartbeat
- Next action
- Key files
- Restart note

## CHECKLIST.md
- [ ] Step 1
- [ ] Step 2
- [ ] Verification

## DOCS.md
- Goal
- Important decisions
- Files that matter
- Gotchas and failure modes
- Notes for the next session

## Rules
- Keep it short.
- Add a heartbeat to active tasks.
- Never leave a folder with only one of the three core notes.
